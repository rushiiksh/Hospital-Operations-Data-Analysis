# Hospital Operations Data Analysis

## Project Overview
This project analyses hospital operations and patient data to uncover trends, evaluate resource utilisation, and provide actionable insights for improving operational efficiency and patient care.

## Objectives
- Analyse patient demographics and medical conditions
- Study billing patterns and hospital costs
- Evaluate room utilisation and admission trends
- Analyse patient length of stay
- Provide business recommendations for hospitals

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git & GitHub

## Project Structure

| Folder    | Description |
|-----------|-------------|
| data      | Raw and cleaned hospital datasets |
| charts    | Visualisations generated during analysis |
| notebooks | Jupyter notebook with complete analysis |
| reports   | Final report and documentation |
| README.md | Project overview and instructions |

## Key Analysis Performed
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Billing Analysis
- Admission Type Analysis
- Room Utilisation Analysis
- Correlation Analysis
- Outlier Detection
- Business Recommendations

## Key Findings
- Patient demographics show diverse healthcare demand
- Billing patterns vary across medical conditions
- Certain rooms experience higher utilisation
- Admission types require operational flexibility
- Length of stay impacts hospital efficiency

## Business Impact
This analysis supports hospitals in improving operational visibility, resource allocation, and cost optimisation while enhancing patient care quality.
